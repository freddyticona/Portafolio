# 📝 Plantilla de Artículos de Blog SEO-Friendly

Guía para crear artículos de blog optimizados para SEO en el portafolio de Freddy Ticona.

---

## 🎯 Estructura de Artículo Optimizado

### 1. Título H1 (Principal)

**Requisitos:**
- 50-60 caracteres máximo
- Incluir palabra clave principal al inicio
- Ser atractivo y descriptivo
- Incluir ubicación si es relevante (La Paz, Bolivia)

**Ejemplos buenos:**
```
✅ "Técnicas de Filmación 4K para Eventos en La Paz"
✅ "Cómo Editar Documentales: Guía para Camarógrafos"
✅ "5 Tips para Cobertura Periodística en Vivo"
```

**Ejemplos malos:**
```
❌ "Mi artículo sobre filmación" (muy genérico)
❌ "Todo lo que necesitas saber sobre edición de video y más..." (demasiado largo)
❌ "Filmación 101" (sin ubicación, poco específico)
```

---

### 2. Meta Description

**Requisitos:**
- 150-160 caracteres máximo
- Incluir palabra clave principal
- Ser persuasivo para el clic
- Resumir el contenido del artículo

**Plantilla:**
```
"Descubre [tema principal] para [audiencia objetivo]. Aprende [beneficio clave] con [número] consejos prácticos basados en [años] años de experiencia en [ubicación]."
```

**Ejemplo:**
```
"Descubre las mejores técnicas de filmación 4K para eventos sociales en La Paz. Aprende composición, iluminación y audio con 5 consejos prácticos basados en 15 años de experiencia."
```

---

### 3. Palabras Clave (Keywords)

**Estructura recomendada:**

**Palabra clave principal:** Una frase principal para enfocar el artículo
- Ejemplo: "técnicas de filmación 4K"

**Palabras clave secundarias:** 3-5 variaciones relacionadas
- filmación de eventos La Paz
- configuración de cámara 4K
- tips de cinematografía
- grabación profesional
- producción audiovisual Bolivia

**Long-tail keywords:** Frases específicas de bajo volumen pero alta intención
- cómo filmar eventos sociales en Bolivia
- mejores ajustes de cámara para bodas
- equipos de filmación para camarógrafos

---

### 4. Estructura de Contenido

```
H1: Título Principal del Artículo

  Introducción (2-3 párrafos)
  - Hook: Captar atención del lector
  - Problema: Desafío que enfrenta el lector
  - Solución: Qué aprenderá en el artículo

  H2: Sección Principal 1
    - Contenido detallado
    - Ejemplos prácticos
    - Imágenes/diagramas

  H2: Sección Principal 2
    - Subtítulo H3: Detalle específico
    - Lista con viñetas para puntos clave
    - Consejos prácticos

  H2: Sección Principal 3
    - Casos de uso reales
    - Ejemplos de proyectos
    - Conclusiones

  H2: Conclusión
    - Resumen de puntos clave
    - Call-to-action (CTA)
```

---

### 5. Imágenes y Multimedia

**Requisitos de imagen:**
- Formato: WebP preferiblemente
- Tamaño: Menos de 500KB
- Dimensiones: 1200x630px para OG, 800x450px para contenido
- Alt text: Descriptivo con palabras clave

**Ejemplo de alt text:**
```
❌ "imagen1.jpg" o "foto"
✅ "Camarógrafo profesional filmando evento 4K en La Paz Bolivia"
```

**Texto de copyright:**
```
Foto: Freddy Ticona - Cobertura de evento corporativo, La Paz 2024
```

---

### 6. Internal Linking

**Enlaces internos a incluir:**
- 2-3 enlaces a otros artículos del blog
- 1 enlace a portafolio relacionado
- 1 enlace a página de contacto (CTA)

**Anchor text (texto del enlace):**
- Usar palabras clave, no "click aquí"
- Ejemplo: "ver técnicas de iluminación" en lugar de "click para más"

---

### 7. Call-to-Action (CTA)

**Al final del artículo:**

```html
<div class="blog-cta">
  <h3>¿Necesitas un profesional para tu próximo proyecto?</h3>
  <p>Con más de 15 años de experiencia en filmación y edición, puedo ayudarte a capturar tus momentos especiales.</p>
  <a href="#contacto" class="btn-primary">Solicitar Cotización</a>
</div>
```

## ✅ Checklist Antes de Publicar

- [ ] Título tiene 50-60 caracteres
- [ ] Meta description tiene 150-160 caracteres
- [ ] Palabra clave principal en título y primer párrafo
- [ ] H2 y H3 organizan el contenido lógicamente
- [ ] Al menos 3 imágenes con alt text descriptivo
- [ ] 2-3 enlaces internos a otras páginas
- [ ] Call-to-action claro al final
- [ ] Longitud mínima: 800 palabras
- [ ] Revisar ortografía y gramática
- [ ] Tags y categorías asignados correctamente
